Sentiment Analysis Techniques
...

  • Applications
    • Mine product reviews
    • Sentiment classification
      • E.g., binary classification for whether a review is positive or negative
    • Track sentiments on topics over time
    • Prediction
      • E.g., election outcomes, market trends
  • Opinion analyzation
    • Opinion aspects
      • Opinion targets
        • Entities and their features
      • Sentiments
        • Whether the opinion is positive or negative
      • Opinion holders
      • Time
        • When the opinion is given
    • Level of detail
      • Document level
      • Sentence level
        • Analyze opinions sentence by sentence
        • E.g., I like the iPhone. [+] I bought an iPhone. [o]
      • Feature level
        • Analyze opinions on each entity's features
        • E.g., entity -> iPhone
          • Features -> touch screen [+], price [-]
  • Challenges
    • Polarity
      • Decide whether the sentiment is positive, negative, or neutral
      • Can be complex especially with mixed opinions
    • Target identification
      • Figure out what the sentiment is about can be tough
        • E.g. when idioms are used
Sentiment Classification
...
  • Statistical association
    • How likely words are to appear near each other
      • Can give clues about sentiment
    • E.g., "unfortunately" is often found near negative sentiments
  • Turney's method
    • Used to automatically classify reviews in an unsupervised way
    1. Phrase extraction
      • Use POS tags to extract two-word phrases
        • E.g., an adverb followed by an adjective and the third word is not a noun
    2. Semantic orientation estimation
      • Point mutual information (PMI)
        • Positive PMI means they co-occur more frequently than would be expected under an independence assumption, vice versa
      • Semantic orientation (SO)
        • Positive value means positive sentiment, vice versa
  • Domain adaption
    • Adapt sentiment analysis tools from one domain to another
    • Pivot features
      • Words or phrases that are important for sentiment
      • And occur frequently in both domains
    • Non-pivot features
      • Indicative of sentiment in the source domain
      • But don't directly correspond to the target domain
    • Goal is to identify
      • Which aspects of sentiment expression are common across domains
      • And which are specific to each one
  • Limitations
    • Domain-dependent polarity
    • Indirect opinion
      • Some sentences express opinion without using explicit opinion words
    • Comparative opinion
      • Expressing a preference for one thing over another
    • No explicit opinion
      • Sometimes, a fact can imply an opinion
Opinion Lexicon Generation
...
  • Create opinion lexicon
    • Find relevant words, phrases or patterns that show subjectivity (personal feelings)
    • Decide if these expressions are
      • Positive, e.g., honest
      • Negative, e.g., harmful
      • Subjective, e.g., curious
  • Hatzivassiloglou & McKeown
    • Adjectives linked by conjunctions have the same or opposite orientation
      • Same orientation
        • E.g., the room was very nice and clean
      • Opposite orientation
        • E.g., the room was very nice but very cold
    • Steps
      1. Label all adjectives with frequency > 20
      2. Extract all adjectives connected by conjunctions
      3. Build a weighted graph and split into two clusters (positive and negative)
Opinion Target Extraction
...
  • Identify what specific aspects of a target entity people have opinions about
  • Example
    • The pictures coming out of this camera are amazing
    • The 'auto' feature takes great pictures most of the time
    • Target entity -> camera
    • Aspects -> pictures, 'auto'
  • Target entity identification
    • Start with a set of seed entities of class , and a set of candidate entities
    • Determine which of the entities in belong to
  • Aspect identification
    • Frequency-based approach
      • Find nouns or noun phrases that are frequently talked about
Advanced Techniques
...
  • Multimodal sentiment analysis
    • Use multiple modalities
      • Text, audio, video
    • Use neural networks to process data and extract features
    • Fuse the features for classification
  • LLM vs SLM
    • LLMs perform better for simple sentiment analysis tasks
    • Smaller language models (SLM)
      • Fine-tuned on domain specific data
      • Performs better for complex sentiment analysis
  • Approaches
    • Zero-shot learning
      • Include only essential components
        • Task name, task definition, output format
      • No examples given
    • Few-shot learning
      • An additional examples are included for each class

Hate Speech and Offensive Language
...
  • Hate Speech
    • A language used to express hatred towards a targeted individual or group
      • On the basis of attributes like race, religion, sexual orientation, etc.
  • Offensive speech
    • Use rude or vulgar language to insult a targeted individual or a group
  • Abusive language classification
    • Reasons for doing it
      • Help places like schools and companies handle hate speech
        • Due to the rules they have to follow
      • Help people moderate online spaces
    • Approaches
      • Social media
        • Use a mix of rules, human moderators, and reports from users
        • Could be a lot of work
      • Automatic
        • Keyword-based
          • Use a list of keywords to identify
          • Low in recall and precision
        • Source metadata
          • Look at the user's details on social media
          • Could lead to privacy/ethical issues and limit freedom of speech
        • ML approaches
HateXplain
...
A set of data (over 20,000 examples) collected from social media sites Twitter and Gab
  • **Annotation
    • Label
      • Hatespeech, offensive, normal
    • Target
      • Groups that could be the subject of hate speech
    • Rationale
      • The specific part of the text that justifies the label
  • Evaluation metrics
    • Performance
      • Accuracy, F1-score, and AUROC measure how well the model identifies hate speech
    • Bias
      • Assess whether the model is biased against certain groups
    • Explainability
      • Plausibility
        • Check if the explanations match what we know
      • Faithfulness
        • Check if they represent why the model made the decision
Freedom of Speech and Counter Hate Speech
...
  • Counter hate speech
    • Responses to hate speech that aim to contradict or oppose it
    • Twitter example
      • One-to-one
        • When a single user engages with another user who has made a hateful statement
      • Many-to-one
        • Multiple users respond to a single harmful tweet
      • One-to-many
        • A single user responds to many others
      • Many-to-many
  • Detoxification
    • Reduce the offensive social media posts while keeping the non-offensive parts intact
      • Rephrases text to eliminate offensive meanings
